# Studio REC website

## Requerimientos
Versiones más recientes de:
 - [Virtualbox][1]
 - [Vagrant][2]

## Instalación
  1. Clonar repositorio
  ```{r, engine='bash'}
  $ git clone https://github.com/dreglad/studiorec
  ```

  1. Instalar ambiente:
  ```{r, engine='bash'}
  $ cd studiorec
  $ vagrant up
  ```

  [1]: https://www.virtualbox.org/ "Oracle Virtualbox"
  [2]: https://www.vagrantup.com/ "Vagrant"
